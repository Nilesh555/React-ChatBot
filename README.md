# 🤖 SmartAssist – Smart Web Assistant

SmartAssist is an AI-powered web chatbot built with React.js that provides intelligent, real-time conversational assistance. It delivers fast, interactive, and user-friendly responses using modern AI APIs, making it suitable for customer support, learning, productivity, and general question answering.

---

## 🚀 Features

- 💬 AI-powered conversational chatbot
- ⚡ Real-time response generation
- 🎨 Modern and responsive React UI
- 🧠 Context-aware conversations
- 📱 Mobile-friendly design
- 🌙 Clean and intuitive user experience
- 🔒 Secure API integration using environment variables
- ♻️ Fast and lightweight performance

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/SmartAssist.git
```

### Navigate to the project

```bash
cd SmartAssist
```

### Install dependencies

```bash
npm install
```

### Add Environment Variables

Create a `.env` file in the project root.

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```

---

## ▶️ Run the Project

```bash
npm run dev
```

The application will start on:

```
http://localhost:5173
```

## 💡 How It Works

1. User enters a question.
2. React captures the message.
3. Request is sent securely to the Gemini API.
4. AI generates a contextual response.
5. SmartAssist displays the response instantly.
